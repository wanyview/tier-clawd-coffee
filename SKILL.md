# TIER_Coffee - 每日咖啡推荐技能

*技能版本: v1.0*
*创建时间: 2026-03-04*

## 技能简介

根据用户每日工作状态，自动推荐适合的咖啡。

## 功能

- 自动获取用户状态（时段、精力、任务完成数）
- 智能推荐咖啡（品牌+产品）
- 16个品牌、46款产品

## 启动方式

```bash
cd ~/clawd
python3 coffee_happ.py
```

## 访问地址

- 主页: http://127.0.0.1:18801
- API: http://127.0.0.1:18801/api/coffee/auto

## 配置

端口: 18801 (可在 coffee_happ.py 中修改)
